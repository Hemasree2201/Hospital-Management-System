# Hospital-Management-System
SQL Based Hospital Management System

## Features
- Manage Patients
- Manage Doctors
- Book Appointments
- Assigning Rooms
- Database Queries
- Stored Procedures
- Triggers

## Technologies Used
- MYSQL
- Database Design
- ER Diagram

## Tables
Patients
Doctors
Nurses
Employees
Receptionist
Records
Rooms
Bills
Test Report

## Relationship tables
Consults - Doctors & Patients
Assigned - Patients & Rooms
Pays - Patients & Bills
Governs - Nurses & Rooms
Manitains - Receptionist & Records
